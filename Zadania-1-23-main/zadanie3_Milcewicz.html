Cechy modyfikowania formatowania w HTML oraz CSS

Cecha HTML CSS
Sposób stylizacji Bezpośrednio w znacznikach (<b>, <font> itd.) Wydzielone reguły (color: red;)
Separacja treści i stylu Styl mieszany z treścią Oddzielenie stylu od struktury
Elastyczność Ograniczona, każdą zmianę trzeba powtarzać Można stosować style globalnie dla wielu elementów
Zaawansowane efekty Ograniczone Animacje, gradienty, cienie
Łatwość utrzymania Trudne przy dużych stronach Łatwiejsze zarządzanie stylem
Możliwość nadpisywania Brak mechanizmu kaskadowego Kaskadowość i dziedziczenie

Nowe możliwości CSS w porównaniu z HTML

  • Kolory i tła – gradienty, przezroczystość, obrazy tła.
  • Układ strony – Flexbox i Grid Layout.
  • Animacje i przejścia – @keyframes,transition.
  • Pseudo-klasy i pseudo-elementy – :hover, :after.
  • Responsywność – Media queries.
  • Efekty cieni i filtrów – box-shadow,filter: blur().
  • Lepsza kontrola nad czcionkami – @font-face.

Wady CSS

  • Brak pełnej kompatybilności między przeglądarkami.
  • Złożoność kaskadowości – nadpisywanie stylów.
  • Brak pełnej kontroli nad wydrukiem.
  • Zależność od silnika przeglądarki.
  • Starsze wersje nie obsługują zmiennych CSS.
