1. Składnia teoretyczna CSS

    2. Przykład składni CSS: 
    <style>

    body {
        background-color: #f0f0f0; 
        font-family: Arial, sans-serif;
    }
    p {
        color: blue; 
        font-size: 16px; 
    }
    h1 {
        color: green; 
        text-align: center;
    }
</style>

  2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik jest za to odpowiedzialny oraz ile razy może on występować w dokumencie, jako podłączenie stylu CSS?

    3. Styl zewnętrzny jest wstawiany w <head> i może występować wielokrotnie.

  3. Jak jest najprostsza forma wstawiania stylu zewnętrznego → otoczony ramką (może być tabela jedno komórkowa?

  4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasad musi być zachowana przy większej ilości dołączanych styli?

    5. Dowolną ilość, trzeba dodać element link rel="Stylesheet" /> przed każdym arkuszem

  5. Co się dzieje w przypadku konfliktów? Przez konflikt rozumiemy, gdy te same style są definiowane w różnych arkuszach styli podłączone do tego samego arkusza.

    6. W przypadku konfliktów, gdy te same style są definiowane w różnych arkuszach stylów podłączonych do tego samego dokumentu HTML, stosowane są zasady kaskadowości

  6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    7. Pliki są tworzone, zostaje wpisana odpowiednia składnia, a następnie zostaje jest dodawane roższerzenie .css podczas zapisu

  7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    8. Program jest uruchamiany, dzięki wielu narzędzią tworzenie pliku jest o wiele łatwiejsze, często także programy oferują podgląd na żywo, a następnie plik jest automatycznie zapisywany z roższerzeniem .css

  8. Jakie rozszerzenie posiadają plik będący zewnętrznym arkuszem styli?

    9. Plik zewnętrznego arkusza styli posiada rozszerzenie .css

  9. Jakie są zalety zewnętrznego arkusza styli( dwie – z użyciem wyliczenia?

    1. wszystkie nasze strony w obrębie całego serwisu mogą mieć pewne wspólne cechy
    2. jeśli w ostatniej chwili zdecydujemy się zmienić np. rodzaj czcionki na wszystkich stronach, możemy to zrobić, modyfikując jedynie zewnętrzny arkusz stylów, bez konieczności zmiany każdej strony osobno

  10. Uwagi na temat pisania w CSS.

    1. LINK tylko w head.
    2. Można dołączyć dowolną liczbę arkuszy.
    3. W przypadku konfliktów ważniejsze będą późniejsze deklaracje.
