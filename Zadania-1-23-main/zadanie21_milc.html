Co znaczy skrót ID? Skrót ID oznacza identyfikator elementu.
Jaki jest cel stosowania ID? ID pozwala na zróżnicowanie podobnych elementów i przypisanie im indywidualnych stylów.
Jakie jest znaczenie znaku # Znak # poprzedza nazwę identyfikatora w arkuszu stylów CSS.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID następuje poprzez atrybut id="nazwa" w znaczniku HTML, a w CSS odwołujemy się do niego za pomocą #nazwa.
Uwagi na temat stosowania ID (trzy konieczne wyliczenia). 1. ID może wystąpić tylko raz w całym dokumencie.
2. ID powinno być unikalne dla każdego elementu.
3. W przypadku wątpliwości co do liczby wystąpień lepiej użyć klasy class.
